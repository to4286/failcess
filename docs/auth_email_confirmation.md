# 이메일 인증 설정 (회원가입 후 즉시 로그인)

회원가입 후 바로 로그인되고 프로필 입력 모달이 뜨려면, Supabase에서 **이메일 인증(Confirm email)**을 비활성화해야 합니다.

## 설정 방법

1. Supabase 대시보드 → **Authentication** → **Providers** → **Email**
2. **Confirm email** 옵션을 **비활성화(Off)**로 변경

이메일 인증이 활성화되어 있으면:
- 회원가입 시 세션이 생성되지 않음
- 이메일 확인 전까지 로그인 불가
- "이메일 혹은 비밀번호가 일치하지 않습니다" 오류 발생
